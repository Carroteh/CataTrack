# CataTrack

> Java program to track Cataract patients and their surgeries

## Technologies

- Java
- JavaFX
- SQLite

## Tracking

### General information:

  - Surname
  - Initials
  - Contact 1 & 2
  - ID number
  - Status

### Eye information:

  - Lens
  - VA
  - Surgery:
    - Date
    - Type
    - Place
    - Notes

## Features

- Add patients
- Perform a patient search
- Perform a general search
- Edit patient information
